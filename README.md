# Proyecto CI/CD con Docker, GitHub Actions y Render

## Descripción
Este proyecto implementa un pipeline de CI/CD usando Docker, GitHub Actions y Render.

## Tecnologías usadas
- Docker
- GitHub Actions
- Render
- Python (Flask)

## Flujo CI/CD

1. Se hace push al repositorio
2. GitHub Actions:
   - Construye la imagen Docker
   - La sube a Docker Hub
3. Render recibe el cambio y hace deploy automático

## Imagen Docker
Docker Hub:
https://hub.docker.com/repository/docker/zizarito/hola-mundo
